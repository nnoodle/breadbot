# breadbot
an auxiliary Discord bot.
![Bread's pfp](./bread.jpg)

## installation
clone the repository and run `npm install`.

## setup
create an `env.json` that has your Discord bot's `token` and bot's `clientid`:

```json
{
  "token": "YOUR-BOT-TOKEN",
  "clientid": "YOUR-BOT-ID",
  "datadir": "data/"
}
```

## running
run `npm start`.
